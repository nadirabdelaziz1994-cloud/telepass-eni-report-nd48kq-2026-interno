GITHUB - GUIDA SEMPLICE

1) Crea un repository PUBLIC su GitHub.
2) Carica TUTTO il contenuto di questa cartella nel repository.
3) Vai su Settings > Pages.
4) In Build and deployment scegli:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /docs
5) Salva.
6) Aspetta il primo aggiornamento automatico (tab Actions).
7) Quando è finito, in Settings > Pages comparirà il link del sito.

COME AGGIORNARE OGNI SETTIMANA

A) Metti il file Lista PDV aggiornato dentro:
   input/lista/
   (tienine uno solo)

B) Metti il file Anagrafica aggiornata dentro:
   input/anagrafica/
   (tienine uno solo)

C) Metti TUTTI i file settimana dentro:
   input/report_settimanali/
   Puoi fare anche sottocartelle tipo:
   input/report_settimanali/2026/SETTIMANA 1/
   input/report_settimanali/2026/SETTIMANA 2/

D) Se vuoi file scaricabili per tutti, mettili dentro:
   FILE_UTILI/

E) Su GitHub premi Add file > Upload files e carica i nuovi file.
F) Premi Commit changes.
G) Vai nella scheda Actions e aspetta che finisca.
H) Il sito si aggiorna da solo.

IMPORTANTE
- Dentro input/lista tieni un solo file Excel.
- Dentro input/anagrafica tieni un solo file Excel.
- Dentro input/report_settimanali puoi mettere tutte le settimane che vuoi.
- Il sito pubblicato è pubblico a chi ha il link. Non mettere dati sensibili.
